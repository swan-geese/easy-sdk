# easy-sdk demo

### 项目介绍
easy-sdk-demo是一个基于easy-sdk的demo项目，主要用于演示easy-sdk的使用方法。

### 项目结构
```
easy-sdk-demo
├── easy-sdk-demo                -- demo 例子
├── easy-sdk-common              -- easy-sdk 通用模块
    ├── easy-sdk-common-config   -- 配置相关 
    ├── easy-sdk-common-core     -- 核心处理
    ├── easy-sdk-common-extra    -- 三方扩展    
```
